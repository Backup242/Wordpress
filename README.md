## PHP

- this tag begins a php block

```php
<?php
```

- this tag ends a php block

```php
?>
```


**Creating a variable in php:**

```php
<?php 
 $myname = "Bryan";

?>

```


**Echoing a variable in php:**

```php

<?php 
echo 2 + 2;
 $myname = "Bryan";
?>
<h1>This page is about <?php echo $myname ?><h1>
    <?php echo 5* 5 ?>
<h2>All about <?php echo $myname ?><h2>

```
